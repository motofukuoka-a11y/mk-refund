# Version 5.2 実装内容

- 普通乗車券に割引種別を追加
- 学生割引（営業キロ100km超、2割引）
- 障害者第1種・介護者同行（距離制限なし、5割引）
- 障害者第1種本人単独（営業キロ100km超、5割引）
- 障害者第2種本人単独（営業キロ100km超、5割引）
- 第1種本人に同行する介護者（距離制限なし、5割引）
- 割引適用条件の自動確認
- 割引後発売額を普通乗車券の払戻計算へ反映
- 使用開始後は「未使用区間の普通運賃（割引前）」へ同じ割引を適用
- 割引条件を `data/discount_rules.json` に分離

## 注意

使用開始後の計算は、既存Versionの「未使用区間相当額から220円を控除する」方式を維持しています。
