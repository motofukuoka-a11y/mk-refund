# MK払戻

JR北海道の払戻計算支援Webアプリです。

## 初期版の対応範囲

- 314隣接区間データによる経路検索
- 営業キロ・換算キロの集計
- 普通運賃、自由席・指定席特急料金、グリーン料金の検索
- 使用開始前の基本的な払戻判定
- 券面額・料金、手数料、払戻額、計算式、判定理由、経路明細の表示
- ライト／ダーク表示

## 現在の未対応範囲

- 使用開始後の普通乗車券
- 定期券の使用経過相当額・旬割
- 普通回数乗車券の使用済枚数控除
- 事故払戻
- 列車別・季節別料金
- 各種割引

未対応項目は、誤計算を防ぐため自動計算対象外として表示します。

## GitHub Pages公開

`Settings` → `Pages` → `Deploy from a branch` → `main` / `/(root)` を設定してください。
