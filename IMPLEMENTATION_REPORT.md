# 定期券入力項目・自動運賃計算 修正報告

## 変更内容

- 定期券の「期間」を1箇月〜6箇月から選択可能に変更
- 定期券入力項目を以下の4項目へ整理
  - 期間
  - 有効期間開始日
  - 申出日
  - 券面金額
- 有効期間終了日は「開始日＋期間−1日」で内部算出
- 発駅・着駅の片道普通運賃を既存運賃表から取得
- 往復普通運賃を「片道普通運賃×2」で自動算出
- 単純計算の使用分を「往復普通運賃×使用日数」で自動算出
- 旬割計算と単純計算を比較し、払戻額が多い方を採用
- 結果欄に往復普通運賃、使用日数、単純計算、旬割計算、使用旬数を表示
