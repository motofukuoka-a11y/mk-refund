# 定期乗車券払戻 修正報告

## 変更内容

- 「単純使用計算」を「使用経過計算」へ名称変更。
- 申出日が有効期間開始日前の場合は、`券面金額－払戻手数料220円`で計算。
- 定期運賃マスタ `data/teiki_fare_master.json` を追加。
- 発着駅、経路、営業キロ、定期種別、期間から券面金額を自動算出。
- 自動算出後の券面金額は手動編集可能。
- 片道普通運賃も従来どおり自動算出後に手動編集可能。
- 旬割計算では、3箇月・6箇月を含め、同一区間・同一定期種別の1箇月定期運賃をマスタから取得して使用。
- 使用経過計算と旬割計算の払戻額を比較し、多い方を採用。
- 2箇月は1箇月×2、4箇月は3箇月＋1箇月、5箇月は3箇月＋1箇月×2として券面金額を自動算出。

## 計算式

### 有効期間開始前

`券面金額－220円＝払戻額`

### 使用経過計算

`券面金額－（片道普通運賃×2×使用日数）－220円＝払戻額`

### 旬割計算

`券面金額－（1箇月定期運賃÷30日・1円未満切上げ×10日×使用旬数）－220円＝払戻額`
