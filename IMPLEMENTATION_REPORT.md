# 定期乗車券払戻 修正報告

## 修正内容

- 「単純計算」の名称と表示を削除し、「単純使用計算」に置換。
- 「有効期間開始日」と「期間（1箇月〜6箇月）」から有効期間終了日を自動算出。
- 有効期間終了日を読み取り専用欄に表示。
- 発駅・着駅・経由駅・旅客区分から片道普通運賃を自動入力。
- 自動入力された片道普通運賃は手動編集可能。
- 単純使用計算を次式で実装。
  - 券面金額 −（片道普通運賃 × 2 × 使用日数）− 220円
- 単純使用計算と旬割計算を比較し、払戻額が多い方を最終払戻額として採用。

## 有効期間終了日

有効期間開始日 + 選択期間（月数）− 1日で算出。月末日は実在する日付へ補正する。
