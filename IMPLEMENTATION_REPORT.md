# MK払戻 初期実装

## 対応券種

- 普通乗車券
  - 使用開始前
  - 使用開始後（未使用区間101km以上、未使用区間運賃を入力）
- 特急券
  - 自由席特急券
  - 指定席特急券
  - 指定席特急券＋グリーン券
  - 座席未指定券
- 普通回数乗車券
  - 発売額－（使用済枚数×券面区間の片道普通運賃）－220円
- 定期乗車券
  - 単純計算：券記載額面－使用分－220円
  - 旬割計算：券記載額面－（日割額×10日×使用旬数）－220円
  - 2方式のうち払戻額が多い方を表示

## 定期券の入力

「単純計算で差し引く使用分」は、月割等で別途算出した使用分を入力します。
旬割側は有効期間、申出日、定期期間、券記載額面から自動計算します。
